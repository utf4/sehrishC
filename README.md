# Manage Bunk Dev

Manage Bunk Dev is a Mean Stack application build to calculate expense of travellers.

## Installation

#### Pre Requisites:

- NodeJS 16.0.0

#### Install

```
npm run install-all

```

## Usage

Run tests (optional)

```
npm run test
```

Start server and client (from root)

```
npm run start

```

#### E2E test cases Cypress

```
npm run e2e

```
